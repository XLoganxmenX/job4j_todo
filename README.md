# *job4j_todo*

job4j_todo - веб-приложение планирования задач. Релизован на языке Java с использованием фреймворка
Spring (Boot + MVC). Фронтенд реализован на Bootstrap.

### Основной функционал приложения:
-  Создание задач с подробным описанием и статусом выполнения;
-  Редактирование или удаление уже созданной задачи;
-  Просмотр всех созданных задач и фильтрация по статусу выполнения.

**Стек проекта:**
- **Язык**: Java 17
- **Сборка проекта**: Maven 3.9.6
- **Фреймворк**: Spring Boot
- **База данных**: PROD - PostgreSQL 16.2 / TEST - H2 Database
- **Миграции БД**: Liquibase
- **ORM**: Hibernate
- **Шаблоны представлений**: Thymeleaf
- **Фронтенд**: Bootstrap

**Требования к окружению**: Java 17, Maven 3.8 или выше, PostgreSQL 14 или выше.

**Приложение состоит из следующих страниц:**
1) <u>Главная страница</u>

    ![Title.PNG](https://github.com/XLoganxmenX/job4j_todo/raw/master/files/Title.PNG)
    
    Представлена краткая информация о сервисе. Слоган и кнопка перехода к странице задач.
2) <u>Страница со списком задач</u>

    ![TaskListPage.PNG](https://github.com/XLoganxmenX/job4j_todo/raw/master/files/TaskListPage.PNG)

    На странице представлен список всех задач пользователя с информацией о названии задачи, дате создания и её статусе.
Можно отфильтровать задачи по статусу или добавить новую задачу. При этом в каждую задачу можно перейти, нажав на неё. В таком случае открывается страница задачи. 
3) <u>Страница задачи</u>

    ![TaskPage.PNG](https://github.com/XLoganxmenX/job4j_todo/raw/master/files/TaskPage.PNG)

    Представлена полная информация о задачи, а именно: название, статус, описание и дата создания.
При нажатии на кнопку "Выполнено", задача переводится в соответствующий статус. Нажав на кнопку "Удалить", задача безвозвратно удаляется из сервиса.
Кнопка "Редактировать" переводит пользователя на страницу для редактирования задачи. 
4) <u>Страница редактирования задачи</u>

   ![EditTaskPage.PNG](https://github.com/XLoganxmenX/job4j_todo/raw/master/files/EditTaskPage.PNG)

   На странице можно поменять название или описание задачи. Сохранить изменения или вернуться обратно.
5) <u>Страница создания задачи</u>

   ![CreateTaskPage.PNG](https://github.com/XLoganxmenX/job4j_todo/raw/master/files/EditTaskPage.PNG)

    На странице пользователь может создать задачу с названием, статусом и полным описанием. Дата создания фиксируется автоматически.   

**Запуск проекта:**

- Установить требуемое в проекте ПО (JDK, Maven, PostgreSQL);
- Создать базу с именем "todo" и параметрами подключения из hibernate.cfg.xml;
- Скачать и распаковать проект; 
- Через терминал перейти в папку проекта и запустить его командой "mvn spring-boot:run";
